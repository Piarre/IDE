# IDE

🚀 A personal CLI for me...

## ⬇️ Installation

```bash
npm install @piarre/IDE
```

## 🚨 Examples

```bash
ide ts -n new-project -d A new project --git
```

## ⚙️ Commands

`ide ts`
<p style="font-weight: bold">Initialize a new TypeScript project (typescript, @types/node, tsup)</p>

```
Options:
  -n, --name <name>                           Project name
  -d, --description <description>             Project description
  -a, --author <author>                       Project author
  -l, --license <license>                     Project license
  -g, --git                                   Initialize git repository
  -dep, --dependencies [dependencies...]      Additional dependencies
  -D, --devDependencies [devDependencies...]  Additional dev dependencies
```
